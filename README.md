# AMS Voice Studio

One-page marketing site for **AMS Voice Studio** (Aaron Snodgrass) in St. Augustine, FL and online. Built to convert visitors into free consultations.

## Live site

Hosted on GitHub Pages: https://marskies.github.io/ams-voice-studio/

## Stack

- Static HTML / CSS / a little JS (no build step)
- Booking via Calendly (popup links)
- Contact form via Formspree (endpoint xpqglgrz)
- Hosted on GitHub Pages
